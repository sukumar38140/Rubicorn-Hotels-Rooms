# 🏨 RUBICORN HOTELS & ROOMS
## Demo Platform

Rubicorn Hotels & Rooms is a premium, full-stack hotel room booking system (Demo Mode) built to showcase a world-class lodging experience for tourists, pilgrims, and business travelers.

---

## 🎨 Design Theme & Aesthetics
- **Style**: Ultra-premium light-cream heritage hotel aesthetic (Oberoi/Taj-inspired).
- **Backgrounds**: Pure white (`#FFFFFF`) and warm light cream (`#FAF7F2`).
- **Typography**: Playfair Display (headings) and Inter (body).
- **Accents**: Deep heritage burgundy (`#7a1f1d`) and warm gold (`#c9a84c`).

## 🛠️ Technology Stack
- **Core**: Vanilla HTML5, CSS3, and modern client-side JavaScript.
- **Routing**: Custom hash-based client-side SPA router (`#/`, `#/rooms`, `#/booking/*`, `#/admin/*`).
- **Icons**: FontAwesome Icons v6.
- **Dev Tooling**: Serves via `http-server` (Dev port: 3000).

## 🚀 Key Features
1. **Interactive Floor Maps**: Select rooms directly from building elevator grids with real-time status feedback.
2. **Flexible Bookings**: Modifiable duration settings (12h/24h stays) and dynamically adjusted pricing.
3. **Responsive Cards**: Tactile scale transitions on hover and active click/touch.
4. **Step-by-Step Wizard**: Guarded booking flow preventing step bypasses until a room is chosen.
5. **Admin Portal**: Floor board console to block rooms for maintenance and process check-outs.

---

## 💻 Running Locally

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Launch Dev Server**:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to [http://localhost:3000](http://localhost:3000).
